# Интернет‑магазин (учебный проект)

## Описание проекта


Это учебный проект интернет‑магазина, разрабатываемый в рамках курса. 

Проект будет поэтапно дорабатываться на каждом занятии. 

В итоге должен получиться полноценный веб‑магазин.

Проект представляет собой веб‑каталог товаров на Django. 

Включает модели Product (товар) и Category (категория), админку с фильтрацией и поиском, миграции, 
фикстуры и кастомную команду для заполнения БД тестовыми данными.


###

## Структура проекта

manage.py — управляющий скрипт Django.

<home_work_shop>/ — корневая папка проекта (настройки, URL‑конфигурация).

catalog/ — приложение для каталога товаров.

catalog/templates/ — HTML‑шаблоны.

catalog/models.py — модели Product, Category, Contact.

catalog/views.py — контроллеры страниц.

catalog/admin.py — регистрация моделей в админке, настройка отображения.

static/ — статические файлы (css, js, image).

fixtures/ — фикстуры для загрузки тестовых данных.

.gitignore — список игнорируемых файлов.

requirements.txt — зависимости проекта.

README.md — описание проекта.


###

### Модели
* Category (категория)

Поля:

name — наименование (строка, макс. 100 символов).

description — описание (текст, макс. 1000 символов).

* Product (товар)

Поля:

name — наименование (строка, макс. 100 символов).

description — описание (текст, макс. 1000 символов).

image — изображение (поле ImageField, загрузка в /image/).

category — связь с Category (ForeignKey, каскадное удаление).

price — цена (десятичное число, 2 знака после запятой).

created_at — дата создания (автоматически).

updated_at — дата последнего изменения (автоматически).

* Contact (контактная информация)

Поля:

phone — телефон.

email — email.

address — адрес.

working_hours — часы работы.



## Установка:

1 Клонируйте репозиторий:
``` git clone git@github.com:lidiar8r-ops/home_work_shop.git  ```

2 Перейдите в папку проекта:
``` cd ваш_репозиторий```

3 Установите зависимости:
``` pip install -r requirements.txt```

4 Примените миграции (создаст таблицы в базе данных):

```
python manage.py makemigrations
python manage.py migrate
```
5 Создайте администратора сайта (чтобы зайти в админку):

```
python manage.py createsuperuser
Введите имя, почту и пароль.
```

6 В терминале запустите сервер:
```python manage.py runserv```

7 Откройте в браузере: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
 
для админки [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)

### Как работать с товарами и категориями
* Через админку (/admin/)

Категории:

Раздел «Категории» → «Добавить».

Заполните «Наименование» и «Описание».

Сохраните.

Товары:

Раздел «Продукты» → «Добавить».

Выберите категорию, загрузите фото, укажите цену и т. д.

Сохраните.

* через кастомную команду, в терминале запустите 
 
```
python manage.py add_products
```
